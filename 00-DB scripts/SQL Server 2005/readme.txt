*** SQL SERVER 2005               ********************************
******************************************************************

Para configurar a base de dados deve efecturar as seguintes operações pela ordem indicada:
1. Criar a base de dados (e.g. Digitarq2) no Sistema de Gestão de Base de Dados (SGBD) SQL Server;
2. Executar as srcipts sql anexadas nesta pasta pela seguinte ordem:
2.1 createTables_2005.sql
2.2 insertRecords.sql 
2.2 storedProcedures_2005.sql
2.3 fulltext.sql
3. Criação dos logins e suas permissões para acesso à base de dados criada em 1.
